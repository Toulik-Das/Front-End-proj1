HTML <small> Tag



Example

Define a smaller text:

<p>Code Academy- the world's largest web development site.</p>
<p><small>Copyright 1960-2050 by Refsnes Data</small></p>
Try it Yourself »

Definition and Usage

The <small> tag defines smaller text (and other side comments).

Default CSS Settings

Most browsers will display the <small> element with the following default values:

Example

small {
    font-size: smaller;
}
